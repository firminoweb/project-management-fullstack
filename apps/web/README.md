# Web — Frontend (React + Vite)

Frontend em **React + Vite + TypeScript** que consome a API de gerenciamento de projetos.

## Telas previstas

- **Listagem de projetos**: nome, status, risco, orçamento, datas e ações (criar, editar, remover, detalhes, análise com IA)
- **Formulário de projeto**: nome, datas, orçamento e descrição (status inicial é automático)
- **Detalhe do projeto**: dados completos + ações de avançar status, cancelar e gerar análise com IA
- **Análise com IA**: resumo, pontos de atenção e recomendação executiva

## Preocupações de UX

- Tratamento de estados de **loading**, **erro** e **lista vazia**
- Validação de formulários
- Exibição clara de **status** e **risco**
